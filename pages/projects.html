DataSmith

: Cross-platform SQL Database Populator for Custom Test Data Generation :

After being tired of writing test@example.com in every project or asking ChatGPT for realistic insert queries, I built DataSmith, a free cross-platform app to generate realistic SQL data with almost no setup.
It auto-detects your DB schema, supports Faker, regex, Python, FKs, and can export .sql or insert directly. Started as a CLI tool: projects DataForge, it evolved into a more user-friendly GUI app with a lot more features built into it.

Skills: Tauri React Rust SQL Python

Healie

: Personal Health Assistant :

  Comes from the words "Health" and "Genie", Healie is a prototype personal health assistant that won the Best Use of AI/ML award from Mr. Najeeb Kanthapuram, MLA (Kerala Legislative Assembly), at the Government of Kerala–backed Scale Up Conclave 2024. It's supposed to work like a health assistant that you can upload your health diagnosis and test data into, from where it would provide a secondary opinion regarding your treatments, as medical misdiagnoses are more common than most people expect.

TheLifeOfPy

: A Simple Artificial Life Simulator :

Inspired by the works of @PrimerBlobs, @davidrandallmiller, and @3Blue1Brown, I've wanted to build something like this for a long time. My BTech final project felt like the perfect excuse to finally do it. So, here's TheLifeOfPy, my attempt at creating a simple artificial life simulation software where you can create organisms by customizing their phenotypic and genotypic traits (like neural networks, size, color, and defense mechanisms) and release them into an environment to see how they interact.

GridNavigatorRE

: Reinforcement Learning based Grid Navigator :

My first Reinforcement Learning project, GridNavigatorRE is a simple grid navigator that uses the DQN algorithm in a custom environment to train an agent to navigate to a fixed goal while avoiding obstacles on the path. I've worked for months on this project and it was pretty rewarding to see the agent stop hitting at the first obstacle it sees and finally navigate to the target.
