DataSmith

: Cross-platform SQL Database Populator for Custom Test Data Generation :

After being tired of writing test@example.com in every project or asking ChatGPT for realistic insert queries, I built DataSmith, a free cross-platform app to generate realistic SQL data with almost no setup. It auto-detects your DB schema, supports Faker, regex, Python, FKs, and can export .sql or insert directly. Started as a CLI tool: projects DataForge, it evolved into a GUI app as I figured others might find it useful too.

GridNavigatorRE

: Reinforcement Learning based Grid Navigator :

My first Reinforcement Learning project, GridNavigatorRE is a simple grid navigator that uses the DQN algorithm in a custom environment to train an agent to navigate to a fixed goal while avoiding obstacles on the path. I've worked for months on this project and it was pretty rewarding to see the agent stop hitting at the first obstacle it sees and finally navigate to the target.

DataForge

: Database Auto-Population tool :

 When I was doing the tedious task of populating databases for testing an API I wrote, I thought, "There's got to be a better way!" And that's how DataForge came to life.

 It's a Python tool I built to auto-generate and insert realistic test data into databases. What's cool about it is how it figures out those foreign key relations to order those tables to fill first. I've had a great time making data insertion a breeze with this project! Hope you find it helpful too.
